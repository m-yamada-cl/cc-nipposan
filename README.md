# cc-nipposan

Claude Codeの開発履歴を自動で日報化するツール

## インストール

```bash
cp -r nipposan ~/.claude/commands/
```

これで以下のスラッシュコマンドが利用可能になります：
- `/nipposan:daily` - 今日の日報を生成
- `/nipposan:all` - プロジェクト全体の日報を生成（注意: 多くのトークンを消費する可能性があります）

## アンインストール

```bash
rm -rf ~/.claude/commands/nipposan
```

## フォルダ構成

```
nipposan/
├── daily.md                    # 日次レポート用スラッシュコマンド
├── all.md                      # 全体レポート用スラッシュコマンド
├── templates/                  # レポート生成用プロンプトテンプレート
└── blog-guidelines/            # ブログ記事執筆ガイドライン
```

## セキュリティとプライバシー

- このツールはローカル環境で動作し、以下のファイルを読み書きします：
  - 読み取り: `~/.claude/projects/` (Claude Codeのセッションログ)
  - 書き込み: `~/Documents/nipposan/` (生成された日報)
- Claude Codeの標準機能によるWeb検索を使用します（ブログドラフト生成時の独自性評価のため）
- すべてのデータはローカルに保存されます

**注意**: 日報にはコード、コミットメッセージ、作業内容が含まれます。
機密情報を含むプロジェクトで使用する場合は、生成された日報の公開前に内容を確認してください。
